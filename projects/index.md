# My Projects

프로젝트 목록 페이지입니다.

## 하위 디렉토리 사용 예시 (Subdirectory Example)

이 페이지는 `projects/index.md` 파일로 생성되었습니다. 
하위 디렉토리를 사용하면 관련 파일들을 함께 구조화할 수 있습니다.

### 디렉토리 구조:
```
projects/
  index.md          # 이 파일 (메인 프로젝트 페이지)
  images/          # 프로젝트 이미지 폴더 (선택사항)
  data/            # 프로젝트 데이터 파일 (선택사항)
```

## 프로젝트 목록 (Project List)

여기에 프로젝트 목록을 추가하세요:

1. **Project 1**: 첫 번째 프로젝트 설명
2. **Project 2**: 두 번째 프로젝트 설명
3. **Project 3**: 세 번째 프로젝트 설명

---

## Tips

- 하위 디렉토리를 사용하면 관련 파일을 함께 관리할 수 있습니다
- 각 프로젝트별로 별도의 페이지를 만들 수도 있습니다
- 이미지나 기타 자료는 같은 디렉토리에 보관하면 편리합니다
